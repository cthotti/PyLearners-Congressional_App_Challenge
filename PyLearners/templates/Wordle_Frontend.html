How to Play:

Wordle instructions: In this game, you must guess the python library or module based on the prompt, which is randomized. After you make a guess by typing in any sequence of characters, different colored boxes for each character indicate how close you were to the answer. A green box means that the character is correct and in the right place, while a yellow box means that the character is a part of the target string but in the wrong place. A grayed out box means that the character is not in the original string. You get as many attempts as there are characters in the target string (for example, if the target string is "random", then users are given 6 attempts to correctly guess the word).
