About PyLearners

How it works: there are 3 different pages on the website, including the wordle game, the function game, and the project dispenser.

    Wordle: by default, the website leads to this page, but you can also click on the sidebar on the left to navigate to this page. In this game, users try to guess the name of a python library in 6 tries, by inputting letters into the prompt boxes and hitting enter. Whether they correctly manage to guess the word or not, they are given a description of the library, which will help them to both learn about new libraries and remember the uses for these libraries.
    Pythonle: this game can also be found on the left sidebar, and the aim of this game is to correctly guess a mystery function in 6 attempts. Users can test the function on the right sidebar by testing out inputs and seeing what they get for the output. They can write out what they think the function is in the textbox with Python code, and hitting submit runs that code through the backend engine. If there is an error, it will not be counted as an attempt, but rather users are shown what their error was.
    Project dispenser: One of the most fulfilling experiences in coding is creating your own functioning program. However, this can be challenging for many less experienced coders. The Project Dispenser solves this problem as not only does it provide an interesting new project, it also comes with clear tutorials carefully selected by our development team. Users can pick a difficulty level (1-3) to randomly generate a new project and begin coding!

Purpose of PyLearners

Our purpose is to create fun and interactive python activities for users to practice The app's target audience is people who are familiar with python but want to improve.
